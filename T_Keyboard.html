การรับค่าจาก Keyboard

นี้ก็เป็นเรื่องสุดท้ายที่ผมจะสอนคุณแล้วนะครับไปลุยกันเลย นอกจากการแสดงผลแล้วนั้น การติดต่อกับผู้ใช้ในอีกรูปแบบหนึ่งคือการรับค่า โดยทั่วไปแล้วมักจะเป็นการรับค่า
ทางคีย์บอร์ด ในภาษา Python ถ้าพิมพ์แค่ input() จะเป็นการการรับค่าแบบ String
เช่น
a = input('กรุณาใส่ชื่อ :') # ข้อความในวงเว็บมีไว้เพื่อบอกให้ user รู้ว่าต้องพิมพ์ค่าจากคีย์บอร์ด เมื่อใส่ชื่อแล้วให้กด Enter ที่ คีย์บอร์ด
print('hello :',a)
เมื่อเรากรอกชื่อโปรแกรมก็จะแสดงผลเป็นชื่อเราออกมา

แล้วถ้าอยากได้เป็นinputเป็น integer ละ จะทำยังไง
เช่น
a = int(input('กรุณใส่ค่า: '))
b = a+5
print(b)
จะเห็นได้ว่าถ้าอยากรับค่าเป็น integer ก็แค่ใส่ int นำหน้า input เท่านั้น



ลองทำดูหน่อยสิครับ
ให้รับค่าจากคีย์บอร์ดเป็นเลข 2 ตัว แล้วแสดงผลบวกออกทางหน้าจอหน่อยสิครับ
a = int(('input = '))
b = (input('input = '))
c = a+b

ให้รับค่าจากคีย์บอร์ดเป็นเลขทศนิยม 2 ตัว แล้วแสดงผลคูณออกมาทางหน้าจอหน่อยสิครับ
a = (input('input = '))
b = float(('input = '))

ให้ a รับค่าจากคีย์บอร์ดเป็นแบบ String เมื่อรันใสค่า 10 ลงไป
ให้ b รับค่าจากคีย์บอร์ดเป็นแบบ Integer เมื่อรันใสค่า 15 ลงไป
