Tuple

ทีนี้ผมก็จะมาสอนเรื่อง Tuple กันนะครับ Tuple จะคล้ายกับ List แต่สิ่งที่แตกต่างกันคือ Tuple นั้นเป็นประเภทข้อมูลที่ไม่สามารถเปลี่ยนแปลงได้
การประกาศ Tuple นั้นสามารถทำได้หลายวิธี รูปแบบพื้นฐานของมันคือสมาชิกของ Tuple ทุกตัวจะอยู่ภายในวงเล็บ ()
และคั่นสมาชิกแต่ละตัวด้วยเครื่องหมายคอมมา (,)
เช่น
a = (1, 2, 3, 4) # เป็นการประกาศตัวแปร Tuples แบบธรรมดา
b = 1, 2, 3, 4 #นี่ก็เป็นอีกวิธีในการประกาศตัวแปรแบบ Tuples
c = tuple(['C++', 'C#', 'Java', 'Pyhton']) #สร้างตัวแปรโดยการใช้ฟังก์ชัน tuple() โดยสร้างจาก List
d = tuple(range(1, 10, 2)) #สร้างตัวแปรโดยการใช้ฟังก์ชัน tuple()จากฟังก์ชัน range() หมายความว่าจาก 1 เพิ่มทีละ 2 ถึง 10

print(a) # (1, 2, 3, 4)
print(b) # (1, 2, 3, 4)
print(c) # ('C++', 'C#', 'Java', 'Pyhton')
print(d) # (1, 3, 5, 7, 9)


Tuple slicing
ในการทำงานกับ Tuple เราสามารถแบ่งมันออกเป็นส่วนย่อยๆ ได้ ด้วยวิธีการที่เรียกว่า
Slicing โดยปกติแล้วมันมักจะใช้สำหรับคัดลอกข้อมูลจาก Tuple เดิมเพื่อสร้างอันใหม่ ซึ่งการ slice นั้นจะทำ
ในรูปแบบของ Index ของ Tuple [start:end] มาดูตัวอย่างกันเลย

n = (1, 2, 3, 4, 'Python', 'PHP', 'Java', 'Thailand', 'Hello')

a= n[0:5] #เริ่มที่ 0 ถึง 5
b= n[4:7] #เริ่มที่ 4 ถึง 7 (7ไม่นับ)
c= n[7:9] # เริ่มที่ 7 ถึง 9 (9ไม่นับ)

print(a) # (1, 2, 3, 4, 'Python')
print(b) # ('Python', 'PHP', 'Java')
print(c) # ('Thailand', 'Hello')

ถ้าหากอยากเปลี่ยนแปลงข้อมูลใน Tuples ละ
myTuple = (1,2,3)

myList.append(4)
