Functions

เรามาต่อกันที่ฟังก์ชันกันเลยนะครับ (Function) คือส่วนของโค้ดหรือโปรแกรมที่ทำงานเพื่อวัตถุประสงค์บางอย่าง ในภาษา Python คุณสามารถสร้างฟังก์ชันของ
คุณเองเพื่อให้ทำงานที่ต้องการในรูปแบบของการประกาศฟังก์ชันในภาษา Python นั้นจะใช้คำสั่ง def และหลังจากนั้น function_name เป็น
ชื่อของฟังก์ชัน และในวงเล็บ () เป็นการกำหนดพารามิเตอร์ของฟังก์ชัน พารามิเตอร์ของฟังก์ชันนั้นสามารถมีจำนวนเท่าไหร่ก็ได้หรือไม่มีก็ได้
ฟังก์ชันอาจจะมีหรือไม่มีการส่งค่ากลับ(return)ก็ได้

return หมายถึง การส่งค่ากลับที่ส่งกลับมาให้ฟังก์ชัน
Parameter หมายถึง ตัวแปรที่กำหนดขึ้นมาเพื่อรองรับค่าของ Arguments ที่จะส่งมาให้กับ Method
Arguments หมายถึง ค่าคงที่หรือตัวแปรต่างๆ ที่เราจะส่งไปให้ Method ที่มีการรับพารามิเตอร์

โครงสร้างฟังก์ชันคือ
def function_name(parameters):
      statement(s)


มาดูตัวอย่างแบบไม่มี Parameter กันเลย
def Test():
      print("Hello Python")

Test() #<<< เป็นการเรียกใช้ฟังก์ชัน Test
ผลรันจะได้ Hello Python
เป็นการประกาศ Function ชื่อว่า Test และไม่มีพารามิเตอร์ ใน Function มีคำสั่ง print ซึ่งจะแสดงค่าออกทางหน้าจอ
เวลาเรียกใช้ Function ก็แค่พิมพ์ชื่อ Function ก็คือ Test() นั้นเอง

ตัวอย่างแบบมี Parameter
def Test2(a, b):
      print(a+b)

Test2(12,45) #เมื่อฟังก์มี parameters(ตัวรับ) ก็ต้องมี Arguments(ตัวส่ง) ในที่นี้คือส่งค่า 12 กับ 45 ให้ฟังก์ขัน Test2
ผลรันจะได้ 57

ตัวอย่างที่มีการ return ค่า
def Test3(a,b):
      c = a+b
      return c

a = Test3(2,3) #Test3 จะนำค่า c เข้าไปเก็บใน a
print(a)
