Simple Mathematical Operators

เอาละครับตอนนี้เราเดินทางมาถึงตัวดำเนินการกันบ้างแล้ว ตัวดำเนินการคำนวณพื้นฐาน มีดังนี้

+ ตัวดำเนินการ บวก
- ตัวดำเนินการ ลบ
* ตัวดำเนินการ คูณ
/ ตัวดำเนินการ หาร
% ตัวดำเนินการ มอดูลัส #หรือก็คือการ mod เลข
** ตัวดำเนินการ ยกกำลัง
// ตัวดำเนินการ หารไม่เอาเศษ

เช่น
a = 5
b = 6
print(a+b) # 11

สามารถเขียนแบบนี้ได้ด้วยนะ
a,b = 5,6
print(a+b)

b = 10
c = 5
d = b ** c
print(d) #d = 100000

a += 5 # หมายถึง a = a+5
ตัวอย่าง
a = 3
b = 4
a += b #เป็นการเปลี่ยนค่าใน a โดยบวก b เพิ่มเข้าไป
print(a) #a = 7
