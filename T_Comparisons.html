Comparisons

เอาละครับทีนี้ก็มาถึงเรื่อง Comparisons กันบ้าง
ในการเปรียบเทียบระหว่างตัวแปร 2 ตัว เราจะใช้เครื่องหมายเท่ากับ2 อัน(==) เทียบว่าทั้ง 2 ตัวแปรเท่ากันหรือไม่ เช่น
a = 5
b = 5
c = b==a
print(c) #True

และเครื่องหมาย ไม่เท่ากัน จะใช้ เครื่องหมายตกใจและเครื่องหมายเท่ากับ(!=)
a = 5
b = 2
c = b!=a
print(c) #True

หากตัวแปรเก็บค่าเป็นตัวอักษร(String) จะไม่สามารถใช้ (==) และ (!=) ในการเทียบกันได้
ต้องใช้ is กับ is not เช่น
a = 'PythonMan handsome'
b = 'PythonMan handsome'
print(a is b) #True #หลักการเท่ากันทุกตัวต้องเหมือนกัน

และกรณีไม่เท่ากัน
a = 'PythonMan handsome'
b = 'PythonMan_handsome'
print(a is not b) #True
