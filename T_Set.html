Set

มาถึงเรื่อง set กันบ้างนะครับ ข้อสังเกตตัวแปรแบบเซตจะมีปีกกาอยู่เสมอ เช่น

a = {1,2,3,4,5,6,'hello'}
b = { 1,2,3,4,123, 'hello', 'python3'}

ตัวแปรแบบเซตสามารถเก็บค่าได้ทั้งตัวเลขและข้อความ
หากอยากเปลี่ยนค่าในเซตต้องใช้คำสั่งต่าง ๆ ดังนี้
การเพิ่มค่าใน a
a.add(ค่าหรือข้อความที่อยากเพิ่ม) เช่น
a.add(10)
print(a) #{1, 2, 3, 4, 5, 6, 10, 'hello'}

การลบค่าใน b
b.discard('hello')
print(b) #{1, 2, 3, 4, 123, 'python3'}

การยูเนียนต่างๆ
print(a.union(b)) #{1, 2, 3, 4, 5, 6, 'python3', 'hello', 123}
print(a.intersection(b)) #{1, 2, 3, 4, 'hello'}
print(a.difference(b)) #{5, 6}
