Dictionary

Dictionary คือประเภทข้อมูลที่เก็บข้อมูลในรูปแบบคู่ ซึ่งจะเป็นรูปแบบเป็น key: value
เช่น
d = {} # empty dict
d = {'key': 'value'} # dict with initial values

เช่น
a= {1: 'One', 2: 'Two', 3: 'Three'}
b= {'One' : 1,'Two' : 2, 'Three' : 3 }

จากตัวอย่าง เราได้ประกาศตัวแปร Dictionary ที่มีชื่อว่า a และ b และมี key: value ตามลำดับ
ซี่ง key ใช้เข้าถึงข้อมูลที่เป็น value และ key เป็นได้ทั้ง String และ int

print(b) # {'Two': 2, 'Three': 3, 'One': 1}
print(b['One']) # 1
print(a.keys()) # dict_keys([1, 2, 3])
print(a.values()) # dict_values(['One', 'Two', 'Three'])


การค้นหาข้อมูลใน dict ด้วย for
for k in a.keys():
print(k, end = ' ') # 1 2 3 (end = ' ' จะทำให้ผลรันเรียงกันไปด้านข้าง)

for k in a.values():
print(k, end = ' ') # One Two Three


การเล่นกับค่าใน ditc เช่น การเพิ่มข้อมูล
a.update({4:'four'})
print(a) #{1: 'One', 2: 'Two', 3: 'Three', 4: 'four'}

การแทนที่
a.update({1:'first'})
print(a) #{1: 'first', 2: 'Two', 3: 'Three'}
