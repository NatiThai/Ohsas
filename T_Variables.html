Variables

มาถึงเรื่องการสร้างตัวแปรในภาษาไพธอนกันนะครับ ในภาษาไพธอนไม่ยุ่งยากในการสร้างตัวแปร เช่น a = 10 ตัวแปร a จะกลายเป็นตัวแปรประเภท intiger(เก็บค่าเป็นตัวเลข)ทันที เมื่อเทียบกับภาษาซีแล้ว จะเป็น int a = 10; b = 10.5 ตัวแปร b จะกลายเป็นตัวแปรประเภท float(เก็บค่าเป็นตัวเลขทศนิยม)ทันที เมื่อเทียบกับภาษาซีแล้ว จะเป็น float b = 10.5; c = 'Hello Python' c เป็นตัวแปรประเภท String เพราะเก็บค่าเป็นชุดตัวอักษร


ตัวอย่าง
a = 10
print(a)
b = 10.5
print(b)
c = 'Hello Python'
print(c)
