Conditionals if-else

ทีนี้ผมจะมาสอนเรื่อง เงื่อนไขการตัดสินใจกันบ้าง
ในที่นี้ผมจะสอนการใช้ if-else ในไพธอนกันนะครับ
if condition:
   body
else:
   body

เช่น
a = 10
if a > 5:
    print('YES')
else:
    print('NO')

คำตอบคือ YES เพราะว่า 10 มากกว่า 5 นั้นเอง
ข้อควรระวัง ผมจะขอยวนไปยังเรื่องที่3ที่ผมสอนคุณไปเรื่อง Indentation สังเกตไหมครับว่า if กับ else จะอยู่ชิดฝั่งซ้ายและ print ไม่ได้อยู่ชิดฝั่งซ้าย หากเขียนผิดอาจจะ error ได้นะครับ
ถ้า
a = 10
if a > 5:
    print('YES')
else:
    print('NO')
print('Done')

จากโค้ดจะมีคำอะไรออกมาทางหน้าจอบ้างน่ะ
คำตอบก็คือ YES กับ Done สังเกตว่าคำสั่ง print('Done') อยู่ชิดฝั่งซ้ายหมายความว่ามันไม่ได้อยู่ในเงื่อนไขมันจึงรันของมันปกตินั้นเอง

กรณีที่มีหลายเงื่อนไขเราจะต้องใช้ elif ด้วย เช่น
a,b,c = 1,2,3
if a == 1:
   print(a)

elif b <3:
   print(b)

elif c >0:
   print(c)

else:
   print('null')
หากรันโค้ดนี้จะเห็นว่ามีแค่ 1 ที่โชว์ออกมาทั้งๆที่ กรณีอื่นก็เข้าเงื่อนไขด้วย เป็นเพราะการทำงานของ if กับ elif จะหาเงื่อนไขที่เป็นจริงตัวแรกตัวเดียว

a,b,c = 1,2,3
if a == 1:
   print(a)

if b < 3:
   print(b)

if c >0:
   print(c)

else:
   print('null')

ผลรันจะเป็น 1 2 3 ตามลำดับ เพราะ if จะเช็คทุกกรณี
ดังนั้นสามารถเลือกใช้ if ทั้งหมด หรือ elif ร่วมด้วยก็ได้แล้วแต่จะเลือกใช้
